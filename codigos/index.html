Início do Código

python 
import numpy as np
from qiskit.quantum_info import Statevector
from IPython.display import display, Latex

1. Parâmetros em graus (Modifique aqui!)

python 
# Insira aqui o valor escolhido para θ, dado em graus
theta_deg = 30

# Insira aqui o valor escolhido para φ, dado em graus
phi_deg   = 90

print("Ângulos escolhidos (em graus):")
print(f"θ = {theta_deg} graus")
print(f"φ = {phi_deg} graus")

2. Conversão para radianos

python 
theta = np.deg2rad(theta_deg)
phi   = np.deg2rad(phi_deg)

print("\nÂngulos convertidos (em radianos):")
print(f"θ = {theta:.12f} rad")
print(f"φ = {phi:.12f} rad")

3. Coeficientes (forma canônica)

python 
alpha = np.cos(theta/2)
beta  = np.exp(1j*phi) * np.sin(theta/2)

print("\nCoeficientes (forma canônica):")
print(f"α = {alpha}")
print(f"β = {beta}")

4. Verificação da normalização

python 
norm = abs(alpha)**2 + abs(beta)**2

print("\nVerificação da normalização:")
print(f"|α|² + |β|² = {norm:.12f}")

5. Estado em LaTeX (forma simbólica)

python 
display(Latex(
r"$$|\psi\rangle="
r"\cos\!\left(\frac{\theta}{2}\right)|0\rangle"
r"+e^{i\phi}\sin\!\left(\frac{\theta}{2}\right)|1\rangle.$$"
))

6. Valores numéricos

python 
display(Latex(
rf"$$\theta={theta_deg}^\circ={theta:.6f}\,\mathrm{{rad}},\quad "
rf"\phi={phi_deg}^\circ={phi:.6f}\,\mathrm{{rad}}.$$"
))

7. Forma expandida

python 
beta_r = 0.0 if abs(beta.real) < 1e-12 else beta.real
beta_i = 0.0 if abs(beta.imag) < 1e-12 else beta.imag

display(Latex(
rf"$$|\psi\rangle="
rf"{alpha.real:.6f}|0\rangle+"
rf"\left({beta_r:.6f}{beta_i:+.6f}i\right)|1\rangle.$$"
))

8. Statevector

python 
psi = Statevector([alpha, beta])

print("\nVetor de estado no Qiskit:")
print(psi)

9. Fase relativa

python 
print("\nFase relativa física:")
print(f"φ = {phi:.12f} rad")
print(f"φ = {phi_deg} graus")

Resultado

O código fixa os ângulos \(\theta\) e \(\phi\) em graus, converte-os para radianos e constrói o qubit puro na parametrização canônica da esfera de Bloch:

\[ |\psi\rangle=\cos\left(\frac{\theta}{2}\right)|0\rangle +e^{i\phi}\sin\left(\frac{\theta}{2}\right)|1\rangle. \]

Em seguida, calcula explicitamente os coeficientes \[ \alpha=\cos\left(\frac{\theta}{2}\right) \qquad \text{e} \qquad \beta=e^{i\phi}\sin\left(\frac{\theta}{2}\right), \] verifica a normalização \[ |\alpha|^2 + |\beta|^2 = 1, \] até erro numérico, e exibe o estado em três formas:

  • simbólica em LaTeX;
  • numérica com \(\theta\) e \(\phi\) em radianos;
  • expandida como \(|\psi\rangle = \alpha|0\rangle + (\Re\beta + i\,\Im\beta)|1\rangle\).

Por fim, encapsula o resultado como um Statevector do Qiskit, isto é, o vetor coluna

\[ |\psi\rangle = \begin{pmatrix} \alpha\\ \beta \end{pmatrix}, \]

na base computacional, e imprime a fase relativa física \(\phi\).

Para os valores escolhidos \(\theta=30^\circ\) e \(\phi=90^\circ\), obtém-se \[ \alpha=\cos(15^\circ)\approx 0.965926 \] e \[ \beta=e^{i\pi/2}\sin(15^\circ)=i\sin(15^\circ)\approx 0.258819\,i. \]

Logo, para esse exemplo,

\[ |\psi\rangle \approx 0.965926|0\rangle + 0.258819\,i\,|1\rangle, \]

com \(|\alpha|^2 + |\beta|^2 \approx 1\).

Visualização e exportação da esfera de Bloch

Após a definição do estado quântico \(|\psi\rangle\), é útil representá-lo graficamente na esfera de Bloch, que fornece uma interpretação geométrica completa para estados de 1 qubit.

O código a seguir utiliza a função plot_bloch_multivector da biblioteca Qiskit para gerar essa representação. A figura é exibida na tela, é salva em formato PDF vetorial e é automaticamente disponibilizada para download no ambiente Google Colab.

O nome do arquivo é gerado dinamicamente em função dos parâmetros angulares \(\theta\) e \(\phi\), o que permite a identificação direta do estado representado. Esse procedimento é essencial para documentação, análise e inclusão de figuras em relatórios científicos.

Recomece no Passo 5.

Desenho, salvamento e download automático do PDF

python 
from qiskit.visualization import plot_bloch_multivector
from IPython.display import display
from google.colab import files

# Gera a figura
fig = plot_bloch_multivector(psi)

# Ajusta tamanho para melhor qualidade
fig.set_size_inches(6.5, 5.5)

# Mostra na tela
display(fig)

# Nome automático com os parâmetros
nome_pdf = f"bloch_theta{theta_deg}deg_phi{phi_deg}deg.pdf"

# Salva em PDF vetorial
fig.savefig(nome_pdf, bbox_inches="tight")

print(f"Figura salva como {nome_pdf}")

# Faz download automático no Colab
files.download(nome_pdf)

Resultado

O trecho acima gera a representação do estado \(|\psi\rangle\) definido no código anterior na esfera de Bloch e produz um arquivo em PDF vetorial com essa figura.

Para o caso \(\theta=30^\circ\) e \(\phi=90^\circ\), a representação geométrica mostra um estado localizado no hemisfério norte da esfera, próximo ao estado \(|0\rangle\), com orientação correspondente à fase relativa \(\phi=90^\circ\).

De acordo com a parametrização canônica de um estado puro de 1 qubit,

\[ |\psi\rangle = \cos\left(\frac{\theta}{2}\right)|0\rangle + e^{i\phi}\sin\left(\frac{\theta}{2}\right)|1\rangle. \]

Substituindo os valores escolhidos:

\[ |\psi\rangle = \cos(15^\circ)|0\rangle + e^{i90^\circ}\sin(15^\circ)|1\rangle = \cos(15^\circ)|0\rangle + i\sin(15^\circ)|1\rangle. \]

Como \(\cos(15^\circ)\approx 0.965926\) e \(\sin(15^\circ)\approx 0.258819\), o estado é dominado pela componente \(|0\rangle\), com contribuição menor da componente \(|1\rangle\).

Geometricamente, o ângulo \(\theta=30^\circ\) determina a distância angular do estado em relação ao polo norte da esfera, associado ao estado \(|0\rangle\). Como esse ângulo é pequeno, o estado permanece próximo desse polo.

Já o ângulo \(\phi=90^\circ\) determina a orientação do estado no plano equatorial. Como \(\phi\) é medido a partir do eixo \(x\) e cresce no sentido anti-horário quando visto do polo norte, o estado fica orientado ao longo do eixo \(y\) positivo.
